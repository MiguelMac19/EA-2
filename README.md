## Career Context

This activity was chosen becuase it builds on EA-01 by shifting focus from offense/defense exploration toward
cross-platform systems fluency. A core requirement for IT Support roles. Working
through both Linux and Windows CLI environments in the same session highlighted gaps in
Windows familiarity that are relevant to helpdesk and endpoint support work. The
Cryptography Basics room also directly supports ongoing CompTIA Security+ preparation.
